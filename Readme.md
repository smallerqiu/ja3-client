# Features
 - Proxy support : HTTP ,HTTPS , Socks4 , Socks5, Socks5h
 - Custom headers
 - Custom JA3 string
 - Custom H2 settings
 - Random TLS extension order
 - Custom TLS extension order
 - Custom connection flow
 - Custom header order
 - Custom client identifier (Chrome, Firefox, Opera, Safari, iOS, iPadOS, Android)


# Dependencies
```
golang ^v1.21x
```
